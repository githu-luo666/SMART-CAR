# LQ_CYT4BBminiMBV2 硬件测试项目

## 项目简介

本项目是基于英飞凌 CYT4BB7 微控制器的硬件测试和控制平台，主要用于智能车竞赛和嵌入式系统学习。项目集成了电机控制、图像处理、传感器数据采集等多种功能，采用双核ARM Cortex-M7架构实现高性能实时控制。

## 硬件平台

- **微控制器**: Infineon CYT4BB7
  - 双核 ARM Cortex-M7
  - 代码Flash: 4MB
  - 工作Flash: 256KB  
  - RAM: 758KB
  - 封装: 144引脚
- **晶振频率**: 16.000MHz
- **系统时钟**: SYS PLL配置为 100MHz + 250MHz + 250MHz

## 开发环境

- **IDE**: IAR Systems 9.40.1
- **编程语言**: C语言
- **调试器**: 支持J-Link/SWD调试接口

## 核心功能模块

### 1. 多核架构
- **CM0+核**: 负责系统初始化和启动其他核心
- **CM7_0核**: 主控制核心，处理图像识别、电机控制、PID算法
- **CM7_1核**: 辅助控制核心，处理传感器数据和通信

### 2. 电机控制系统
- **驱动方式**: 支持8701和7971电机驱动芯片
- **控制算法**: 增量式PID控制算法
- **编码器支持**: 正交编码器接口，支持速度和位置反馈
- **PWM输出**: 多路PWM输出控制电机速度

### 3. 图像处理系统
- **摄像头**: MT9V034全局快门摄像头
- **图像分辨率**: 94×60像素 (可配置188×120)
- **处理算法**:
  - 图像二值化 (Otsu算法)
  - 边缘检测 (Sobel算子)
  - 形态学滤波
  - 赛道特征识别 (直线、弯道、十字、圆环)
  - 中线提取和误差计算

### 4. 传感器系统
- **陀螺仪**: 
  - 支持SPI和I2C接口
  - 型号: ICM20689、ICM20602、ICM42605、ICM42688
  - 功能: 姿态解算、角度积分
- **编码器**: 速度测量和位置反馈
- **距离传感器**: VL53激光测距
- **按键**: GPIO按键输入

### 5. 显示系统
- **屏幕支持**: 
  - 1.14寸、1.3寸、1.54寸 IPS屏幕
  - 1.8寸 TFT屏幕
  - 0.96寸 OLED屏幕
- **通信接口**: 支持硬件SPI和软件SPI

### 6. 通信接口
- **UART**: 串口通信，用于调试和数据输出
- **SPI**: 高速串行外设接口
- **I2C**: 低速串行外设接口
- **IPC**: 核间通信，实现多核数据交换

## 主要算法

### PID控制算法
```c
typedef struct PID
{
    long SumError;      // 误差累积
    int32 LastError;    // 上次误差
    int32 PrevError;    // 上上次误差
    float KP;           // 比例系数
    float KI;           // 积分系数
    float KD;           // 微分系数
} PID;
```

### 图像处理流程
1. **图像采集**: 从MT9V034摄像头获取原始图像
2. **二值化**: 使用Otsu算法进行自适应阈值分割
3. **特征识别**: 识别直线、弯道、十字、圆环等赛道特征
4. **误差计算**: 计算赛道中线与理想路径的偏差
5. **控制输出**: 根据误差调整电机PWM输出

### 姿态解算
- **陀螺仪数据采集**: 定时读取加速度计和陀螺仪数据
- **角度积分**: 对角速度进行积分得到姿态角度
- **姿态补偿**: 在弯道和特殊路段进行姿态补偿

## 使用方法

### 1. 编译项目
- 使用IAR Systems 9.40.1打开项目
- 选择对应的编译配置 (Debug/Release)
- 点击编译按钮生成hex文件

### 2. 下载程序
- 连接J-Link调试器
- 选择对应的调试接口 (SWD)
- 下载程序到目标板

### 3. 测试模块
- 在`main_cm7_1.c`中取消注释需要测试的函数
- 重新编译下载
- 观察测试结果

### 4. 运行主程序
- 确保所有测试函数已注释
- 编译下载主程序
- 系统将自动运行图像识别和电机控制

## 注意事项

1. **多核同步**: 注意不同核心之间的数据同步和通信
2. **中断优先级**: 合理设置中断优先级，避免冲突
3. **内存管理**: 注意堆栈大小，避免溢出
4. **时序要求**: 严格满足各模块的时序要求
5. **电源稳定**: 确保电源稳定，避免电压波动影响系统稳定性
